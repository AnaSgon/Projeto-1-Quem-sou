Para iniciar no Front end vamos focar em desenvolvimento do html, para isso é utilizado as semanticas como elementos de marcação do html conhecido como Tags, elas dão estruturação ao contéudo imposto e para meio de treino foi usado o programa VsCode


Como iniciar um projeto no VScode?

  1. Crie uma Pasta na Aréa de Trabalho
  2. Abra o Vscode e Vá em File > Open Folder
  3. Crie quantas paginas precisar em NewFile nomeie "nome.html" - Obs: fica ao lado do nome do projeto
  4. Para abrir o html5 insira "!" e aperte enter

Detalhes para iniciar um codigo

  • Observe a linguagem do site, se inicia em inglês (lang = "en") podendo ser alterada para português (Pt-Br)
  • Aplique a extensão Extensão Live server clicando na pagina sendo editada com o lado direito do mouse e selecione "open with live server"

Modos de Inserir uma imagem:

Baixando e aplicando no html
  • Vá na imagem com o botão direito e selecione "baicar imagem como"
  • Adcione a imagem na pasta onde esta o codigo
  • Vá no codigo e digite "img"
  • Há 2 indicadores: "Src" onde deve inserir o nome da imagem salva "nome.formatoSalvo" e "Alt" onde deve inserir a descrição da imagem
  • IPC: se a imagem tiver dentro de uma outra pasta deve-se informar "NomeDaPasta.NomeDaImagem.Formato"
Através do link da imagem
  • selecionar imagem com o botão direito "copiar endereço da imagem"
  • colar link da imagem entre as aspas de "Src" para "Alt" é o mesmo procedimento

Como alterar o tamanho da imagem

  • Para alterar Altura (Height=X"px")
